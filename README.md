# chatbot

**Для работы гуглового api нужен впн**

1.  **Перейдите в папку с проектом:**

    ```bash
    cd /путь/к/папке/проекта
    ```

2.  **Активируйте виртуальное окружение:**

      * **Для Linux/macOS:**
        ```bash
        source bin/activate
        ```
      * **Для Windows (PowerShell):**
        ```bash
        .\venv\Scripts\Activate.ps1
        ```
      * **Для Windows (CMD):**
        ```bash
        .\venv\Scripts\activate.bat
        ```

    > ⚠️ **Примечание:** Если возникают проблемы с запуском, это может быть связано с настройками **Python** в вашей операционной системе Windows.

3.  **Установите все зависимости** (если это не было сделано ранее):

    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите приложение:**

    Выберите один из следующих вариантов запуска:

      * **С использованием Uvicorn (рекомендовано для разработки с горячей перезагрузкой):**
        ```bash
        uvicorn app:app --reload
        ```
      * **Запуск как обычный Python-скрипт (если это предусмотрено структурой проекта):**
        ```bash
        python app.py
        ```
      * если в ide открыть venv (не спрашивайте как я хз) должно запускаться из ide
